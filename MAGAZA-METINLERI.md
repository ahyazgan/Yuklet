- **Aktarımda şifreleme (in transit):** Evet (TLS)
- **Kullanıcı silme talep edebilir mi:** Evet (uygulama içi + e-posta)
- **Üçüncü taraflarla paylaşım (sharing):** Hayır (yalnızca işleyiciler)
- **Reklam/pazarlama için toplama:** Hayır
- **Toplanmayan:** finansal bilgi (v1'de ödeme yok), sağlık, cihaz rehberi, tarama geçmişi, reklam kimliği

### Apple — App Privacy ("nutrition label")

- **Data Used to Track You:** None (izleme yok)
- **Data Linked to You:**
  - Contact Info: Name, Email, Phone
  - User Content: Photos, Messages, other (ilan içeriği)
  - Identifiers: User ID
  - Location: Precise Location _(yalnızca sefer takibi kullanılırsa)_
  - Diagnostics: Crash/Performance data
- **Purpose (hepsi için):** App Functionality (+ hesap yönetimi). Reklam/izleme yok.
